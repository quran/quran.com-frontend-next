## Content Fonts

### Translation

Uses rem units for consistent scaling.

| Scale | Tablet (rem) | Mobile (rem) |
| ----- | ------------ | ------------ |
| 1     | 0.85rem      | 0.85rem      |
| 2     | 1rem         | 0.95rem      |
| 3     | 1.25rem      | 1.05rem      |
| 4     | 1.4rem       | 1.2rem       |
| 5     | 2rem         | 1.5rem       |
| 6     | 2.3rem       | 1.8rem       |
| 7     | 2.7rem       | 2.1rem       |
| 8     | 3rem         | 2.4rem       |
| 9     | 3.5rem       | 2.75rem      |
| 10    | 4rem         | 3rem         |

### Tafsir

Uses rem units, similar to translations but with slight variations at smaller scales.

| Scale | Tablet (rem) | Mobile (rem) |
| ----- | ------------ | ------------ |
| 1     | 0.85rem      | 0.85rem      |
| 2     | 1.1rem       | 0.95rem      |
| 3     | 1.2rem       | 1.1rem       |
| 4     | 1.4rem       | 1.2rem       |
| 5     | 2rem         | 1.4rem       |
| 6     | 2.3rem       | 1.8rem       |
| 7     | 2.7rem       | 2.1rem       |
| 8     | 3rem         | 2.4rem       |
| 9     | 3.5rem       | 2.75rem      |
| 10    | 4rem         | 3rem         |

### Word-by-Word

Uses CSS custom properties. **Maximum scale is 6** (not 10 like other content types).

| Scale | CSS Variable                   | Actual Value |
| ----- | ------------------------------ | ------------ |
| 1     | `--font-size-xsmall-dangerous` | 0.6rem       |
| 2     | `--font-size-xsmall`           | 0.75rem      |
| 3     | `--font-size-small`            | 0.875rem     |
| 4     | `--font-size-normal`           | 1rem         |
| 5     | `--font-size-large`            | 1.125rem     |
| 6     | `--font-size-xlarge`           | 1.5rem       |

### Reflections

Uses CSS custom properties with a mix of rem and px values.

| Scale | CSS Variable              | Actual Value |
| ----- | ------------------------- | ------------ |
| 1     | `--font-size-xsmall`      | 0.75rem      |
| 2     | `--font-size-normal`      | 1rem         |
| 3     | `--font-size-large`       | 1.125rem     |
| 4     | `--font-size-xlarge`      | 1.5rem       |
| 5     | `--font-size-xlarge-px`   | 26px         |
| 6     | `--font-size-xxlarge-px`  | 32px         |
| 7     | `--font-size-xxxlarge-px` | 36px         |
| 8     | `--font-size-jumbo-px`    | 40px         |
| 9     | `--font-size-jumbo2-px`   | 48px         |
| 10    | `--font-size-xjumbo-px`   | 64px         |

### Lessons

Uses the same mapping as Reflections (scales 1-10 with the same CSS variables).

| 3     | 1.2rem       | 1.1rem       |
| 4     | 1.4rem       | 1.2rem       |
| 5     | 2rem         | 1.4rem       |
| 6     | 2.3rem       | 1.8rem       |
| 7     | 2.7rem       | 2.1rem       |
| 8     | 3rem         | 2.4rem       |
| 9     | 3.5rem       | 2.75rem      |
| 10    | 4rem         | 3rem         |
